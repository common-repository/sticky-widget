== Description ==

Set any widget as a fixed / sticky widget, (remain at the same point while scrolling) by simply checking the option. 



== Installation ==

1. Follow standard WordPress plugin installation procedure
2. Activate the plugin through the Plugins menu in WordPress
3. Go to Appearance -> Widgets, enable "Sticky Widget" option on any active widget 
4. Fine tune fixed widget margins on Appearance -> Sticky Widget Options page
